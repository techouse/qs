Unified scalar decode with key/value context.

Uses a provided custom Decoder when set; otherwise falls back to Utils.decode. For backward compatibility, a LegacyDecoder can be supplied and is honored when no primary Decoder is provided. The kind will be DecodeKind.key for keys (and key segments) and DecodeKind.value for values. The default implementation does not vary decoding based on kind.
